3️⃣ Compliance Mapping Logic

Now your backend decides which regulations apply.

Example logic:

def map_regulations(org_type):

    if org_type == "NBFC":
        return [
            "NBFC KYC Master Direction"
        ]

    elif org_type == "Payment Aggregator":
        return [
            "Regulation of Payment Aggregator"
        ]

    elif org_type == "Commercial Bank":
        return [
            "Commercial Banks KYC Direction",
            "Basel III Capital Regulations"
        ]
